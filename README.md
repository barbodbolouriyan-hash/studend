# Student Performance Prediction

## Overview
This project uses the Student Performance dataset to predict students' academic performance using Machine Learning classification algorithms. The dataset contains information such as age, gender, study time, absences, parents' education, family status, and other factors that may affect student performance.

## Dataset
- Dataset: Student Performance (Mathematics)
- File: student-mat.csv

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook (VS Code)

## Machine Learning Models
- Decision Tree (DT)
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes (NB)

## Evaluation Metrics
- Accuracy
- Precision
- Recall

## Results
The Gaussian Naive Bayes model achieved the best performance with:
- Accuracy: 1.00
- Precision: 1.00
- Recall: 1.00

## Conclusion
Among the evaluated models, Gaussian Naive Bayes achieved the highest performance on this dataset. Therefore, it was selected as the best model for predicting student performance.
